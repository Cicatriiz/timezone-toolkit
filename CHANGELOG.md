# Changelog

All notable changes to the TimezoneToolkit MCP server will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-25

### Added
- Initial release of TimezoneToolkit
- Basic timezone conversion tools
  - convert_time: Convert times between timezones
  - get_current_time: Get current time in any timezone
  - calculate_timezone_difference: Calculate time difference between timezones
  - list_timezones: List available IANA timezones by region
- Astronomical calculation tools
  - calculate_sunrise_sunset: Calculate sunrise, sunset, and twilight times
  - calculate_moon_phase: Calculate moon phase for any date
- Date and time utility tools
  - format_date: Format dates in various styles and locales
  - calculate_countdown: Calculate time remaining until a specific date/event
  - calculate_business_days: Calculate business days between dates
- Comprehensive documentation
- Test suite for all tools
